### Description

**Download Time Calclulator** is a **PWA** to calculate the download time of your files


### Tecnologies

- Nodejs
- Bootstrap 4
- Vanilla js

## Setup

```bash
$ git clone https://github.com/rafieltq/downloadtimeCalculator.git
```

```bash
$ npm install 
```

```bash
$ npm install webpack
$ npm install webpack-dev-server
# to execute a test enviroment
$ npm run server
```
